# MnLua for VS Code

Welcome to MnLua!

## Features

Provide syntax highlighting for Lua scripts in [MiniWorld](https://www.miniworldgame.com).

## Requirements

- Visual Studio Code: `1.78.0`
- Sumneko Lua: `3.6.19`

## Extension Settings

MnLua contributes the following settings:

- `mnlua.enable`: Enable/disable MnLua.
